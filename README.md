# 🛠️ Nasıl Kullanılır?

1. **Uygulama Sayfasına Git:** [Buraya tıkla](https://brkyyrr.github.io/instagram-followers/) ve **KODU KOPYALA** butonuna bas.
2. **Instagram'ı Aç:** [instagram.com](https://www.instagram.com) adresine git ve giriş yap.
3. **Konsolu Aç:** `F12` veya `Sağ Tık > İncele > Console` yolunu izle.
4. **Çalıştır:** Kopyaladığın kodu yapıştır ve `Enter`'a bas.
5. **Analiz:** Açılan panelde kullanıcı adını yaz ve **Analizi Başlat**.

---

# ✨ Öne Çıkan Güvenlik Özellikleri
* **Gelişmiş Anti-Ban:** İşlemler arasına rastgele 25-45 saniye bekleme ekler.
* **Zorunlu Mola:** Her 5 unfollow işleminden sonra 5 dakikalık (300sn) güvenlik molası verir.
* **Gizli Hesap Filtresi:** Listelerinizdeki gizli hesapları kolayca ayırt etmenizi sağlar.
